# Vue3宠物领养后台管理系统 java382

## 项目概述

本项目是一个基于Vue3的宠物领养后台管理系统，采用纯前端技术实现，不涉及后端数据库操作。系统界面美观，功能模块齐全，业务逻辑清晰，适合用于学习和掌握Vue3 Composition API及后台管理系统的开发。

## 技术栈

- **前端框架：** Vue3
- **UI组件库：** Element Plus
- **构建工具：** Vite
- **状态管理：** Pinia
- **路由管理：** Vue Router
- **HTTP请求：** Axios
- **数据可视化：** ECharts
- **日期处理：** Day.js
- **工具库：** Lodash
- **CSS预处理器：** Sass

## 功能模块

系统包含以下11个主要页面：

- 仪表盘首页
- 宠物分类管理
- 宠物信息管理
- 领养申请管理
- 用户管理
- 角色管理
- 权限管理
- 系统设置
- 登录页
- 注册页
- 错误页面

此外，还包括以下自定义组件：

- 15个以上的自定义组件

## 系统角色

系统实施基于角色的权限管理，包括但不限于以下角色：

- 系统管理员
- 宠物信息管理者
- 用户管理者
- 角色与权限管理者

## 运行环境

- 现代浏览器（支持ES6+）

## 部署步骤

1. 安装Node.js环境
2. 使用Git克隆项目代码
3. 进入项目目录，运行`npm install`安装依赖
4. 执行`npm run dev`启动开发环境
5. 执行`npm run build`构建生产环境代码

project/

## 核心亮点

- **纯前端实现：** 无需后端数据库，即可展示完整功能。
- **状态管理：** 使用Pinia实现高效的状态管理。
- **数据可视化：** 集成ECharts，实现数据的直观展示。
- **权限控制：** 基于角色的权限控制系统，确保系统安全。
- **丰富的组件：** 15个以上的自定义组件，提高开发效率。
- **响应式设计：** 页面美观，支持多种设备访问。
- **详细的开发文档：** 方便学习和二次开发。
